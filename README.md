# 🎯 タスクマスター Pro - Advanced Task Management App

## 概要

タスクマスター Proは、最新のCustomTkinterを使用して開発された、美しく高機能なデスクトップ・タスク管理アプリケーションです。ポモドーロタイマー、詳細統計、進捗管理など、生産性を向上させる革新的な機能を搭載しています。

---

## ✨ 主要機能

### 📋 **高度なタスク管理**
- ✅ 直感的なタスクの作成・編集・削除
- 🏷️ タグシステムによる柔軟な分類
- 📊 リアルタイム進捗バー表示
- 🎨 優先度別カラーコーディング
- 📅 期限管理と期限切れアラート
- 📝 詳細な説明とカテゴリ分類

### 🍅 **ポモドーロタイマー機能**
- ⏰ カスタマイズ可能な作業・休憩時間
- 🎵 セッション完了時の通知音
- 📈 ポモドーロ完了回数の自動追跡
- 🔄 作業セッションとタスクの自動連携
- ⏸️ 一時停止・再開機能

### 📊 **詳細統計分析**
- 📈 生産性レポートと完了率分析
- 📁 カテゴリ別パフォーマンス統計
- 🏆 作業効率性の測定
- 📅 週別進捗トレンド
- 🏷️ タグ使用頻度分析
- 📤 統計データのエクスポート機能

### ⚙️ **カスタマイズ設定**
- 🌓 ダーク/ライトテーマ切り替え
- 🔔 通知音のオン/オフ設定
- 💾 データのインポート/エクスポート
- 🎛️ ポモドーロタイマーの時間設定

## 🚀 インストールとセットアップ

### 前提条件
- Anaconda または Miniconda がインストールされていること
- Python 3.11+ 対応

### セットアップ手順

1. **リポジトリのクローン**
   ```bash
   git clone https://github.com/Kota-James/my-task-app.git
   cd my-task-app
   ```

2. **Conda仮想環境の作成**
   ```bash
   conda env create -f environment.yml
   ```

3. **仮想環境のアクティベート**
   ```bash
   conda activate my-task-app-env
   ```

4. **アプリケーションの実行**
   ```bash
   python Src/main.py
   ```

## 📖 使用方法

### 基本操作

1. **新しいタスクの作成**
   - 「➕ 新しいタスク」ボタンをクリック
   - タスク名、説明、優先度、期限、タグ、予想時間を入力
   - 進捗バーで現在の進捗を設定

2. **ポモドーロタイマーの使用**
   - 「🍅 ポモドーロ」タブに切り替え
   - 作業したいタスクを選択
   - 作業時間と休憩時間を設定
   - 「▶️ 開始」ボタンでタイマー開始

3. **統計の確認**
   - 「📊 統計」タブで詳細な分析を確認
   - 生産性、カテゴリ別統計、進捗トレンドを表示
   - データのエクスポートが可能

4. **設定のカスタマイズ**
   - 「⚙️ 設定」タブでアプリをカスタマイズ
   - テーマ、通知、データ管理の設定
